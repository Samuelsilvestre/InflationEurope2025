# Análise e Tratamento de Dados de Inflação (IHPC) dos Países da União Europeia

Este projeto de análise de dados em Python visa processar, limpar e analisar o Índice Harmonizado de Preços ao Consumidor (IHPC, ou HICP em inglês) de diversos países da União Europeia. O objetivo é comparar o comportamento da inflação entre os países que adotam a política monetária do Banco Central Europeu (BCE) e aqueles que possuem Bancos Centrais Nacionais (BCN).

## Fonte dos Dados

Os dados brutos utilizados neste projeto foram obtidos diretamente da base de dados do Eurostat:

* **Dataset:** HICP - monthly data (index)

* **Referência:** Eurostat Databrowser
